# Habiter_App

## Overview:
Habiter is an app that allows the user to create, track, delete habits. The whole purpose behind this app is to promote the user to create good habits and continue to keep those habits through logging their habit daily or weekly. The User is able to view stats on their completed goal and achieve some acheivements based on their completion of that goal.

## Current Features:
- Upload/Insert Habit
- Delete Habit 
- View Stats
- Change Settings
- View Calender
- Edit Habit
